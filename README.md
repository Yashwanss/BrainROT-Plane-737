## 🚀 Installation

This game requires **Python 3.10+**. 

### Quick Install (Windows)
We've included a handy script that automatically installs everything you need to play!
1. Double-click the `install.bat` file in the main folder.
2. Once the script finishes, it will pause. Just press any key to close it.
3. You're ready to play!

### Manual Install
If you prefer using the command line or are not on Windows, you can install the dependencies manually using pip.

1. Open a terminal or command prompt in the project's root folder.
2. Run the following command to install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
